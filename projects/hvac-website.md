# HVAC Website — Client Service Platform

## Overview
A client-facing website for an HVAC services business, built using AI-native development tools to demonstrate rapid product delivery.

## Approach
- Translated business requirements directly into deployed software using Lovable (AI-native development platform)
- Went from written requirements to live production site without traditional dev cycles
- Frontend deployed on Vercel for fast, reliable hosting

## Significance
This project demonstrates the AI-native product development approach — using modern AI tools to go from concept to deployed product at startup speed. The same methodology is applied across all SokoTech client engagements.

## Stack
Lovable · Vercel · AI-Native Development
